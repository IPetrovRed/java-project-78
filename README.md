### Проект "Валидатор данных"
Выполняет задачи по прорверки корректиности вводимых пользователями данных. К примеру, при заполнении различных форм.

## Пример работы программы

# Java package

# Run
```bash
make run
```

## Setup
```bash
make build
```

## Run-dist
```bash
make run-dist
```

## Run test
```bash
make test
```

## Run report
```bash
make report
```

## Run checkstyle
```bash
make lint
```

## Check update dependencies and plugins
```bash
make check-deps
```
